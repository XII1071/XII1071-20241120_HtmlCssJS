operator :: 연산자

Programming에서 Processing을 처리하는 첫번째 연산자

홈으로
  1. 산출 연산자(Arithmetic Operators)

    2. + , - , *, /, %, **

  2. 비교 연산자(Comparison Operators)

    3. ==, ===, !=, >, <, >=, <=

    비교 연산자의 결과값은 ture, false

  3. 논리 연산자

    4. true, false에 대한 &&(논리곱)and, ||(논리합)or, !(부정)

    truthy, falsy 값에 대한 &&, || 연산을 할 경우는 true,false가 아닌 값을 반영

    &&연산자가 ||보다 우선순위가 높다

            자바스크립트에서 truthy value
    1) 0을 제외한 정수와 실수, Infinity, -Infinity
    2) 빈문자열을 제외한 문자열("0")
    3) new Date()와 같은 참조형 객체들
    4) {} 빈 객체, []
    5) function(){} 
    6) true
    7) DOM API에 정의된 객체들(window, document 등등)

    자바스크립트에서 falsy value
    1) undefined, null
    2) NaN(Not A Number)
    3) 0(숫자 리터럴), -0
    4) ""(빈 문자열) 빈문자열은 false로 인식하지만, 빈 객체({})는 true로 인식함.
    5) []는 비교연산자를 만나면 '' 또는 ""로 변환되어서 빈문자열로 인식해서 false.
    6) false
    7) document.all,  alert()는 값을 반환하지 않으므로 undefined를 반환
  4. 단항 연산자

    5. ++, --

    a++, ++a => a = a + 1 / a--, --a => a= a -1

  5. 대입 연산자

    6. =, +=, -=, *=,/=, %=, **=

  6. 삼항 연산자

    7. 조건? 참일경우 : 거짓일 경우

  7. 시프트 연산자

    8. >>, <<
